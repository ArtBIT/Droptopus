# Droptopus
Droptopus is a drag'n'drop target, which routes dropped objects to the designated actions.

## Dependencies
 - PyQt4
 - Python Magic 
 `sudo apt-get install python-pyqt4 python-magic`

## Installation

 # Clone the repo:
 `git clone https://github.com/ArtBIT/Droptopus`
 # Run it:
 `cd Droptopus && ./start.sh`
